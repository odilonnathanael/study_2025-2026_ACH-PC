# Лабораторная работа №4
## Создание и процесс обработки программ на языке ассемблера NASM

### Цель работы
Освоение процедуры компиляции и сборки программ, написанных на ассемблере NASM.

### Теоретическое введение
Язык ассемблера - это машинно-ориентированный язык низкого уровня, который позволяет получить доступ к архитектуре ЭВМ и её аппаратным возможностям.

### Выполнение работы
1. Создан каталог для работы
2. Написана программа Hello World на NASM
3. Программа оттранслирована с помощью nasm
4. Скомпонована с помощью ld
5. Программа успешно запущена
6. Создана модифицированная версия с своим именем
7. Все файлы загружены на GitHub

### Выводы
Освоен процесс компиляции и сборки программ на NASM. Приобретены практические навыки работы с транслятором и компоновщиком.
