Отчёт - Лабораторная работа №7

Команды условного и безусловного переходов в NASM

Фамилия Имя: Лиссон Байем
Дата: 23 ноября 2025 года

1. Цель

Изучить команды условного и безусловного переходов в ассемблере NASM. Приобрести навыки написания программ с переходами. Ознакомиться со структурой и назначением файла листинга.

2. Полученные результаты

2.1 Безусловные переходы (lab7-1.asm)

· Исходная программа с переходом на _label2
· Модификация для порядка 2→1→конец
· Модификация для порядка 3→2→1

2.2 Условные переходы (lab7-2.asm)

· Программа поиска максимального значения среди A, B и C
· Тестирование с различными значениями B
· Использование инструкций CMP и JG

2.3 Файл листинга

· Успешное создание файла lab7-2.lst
· Анализ формата (адреса, машинный код, исходный код)
· Тестирование ошибок: поведение при ошибке компиляции

2.4 Самостоятельная работа

· lab7-3.asm: Поиск минимального значения (вариант 1: 17, 23, 45)
· lab7-4.asm: Вычисление функции (вариант 1) с решением проблем ввода

3. Возникшие трудности и решения

· Проблема с отображением русских символов → временное использование сообщений на английском
· Ошибка компиляции lab7-4.asm → очистка возвратов каретки во входных данных
· Опечатки в командах ld → исправление elf_1386 на elf_i386

4. Вывод

Приобретены навыки программирования переходов в ассемблере NASM. Освоены условные и безусловные переходы, а также генерация файлов листинга для отладки.
